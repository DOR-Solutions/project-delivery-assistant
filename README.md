# Project Delivery Assistant

## Description
A system to assist with project delivery management.

## Getting Started
Instructions to set up and run the project locally.

### Backend
Navigate to the `backend` directory and install the dependencies:
```
cd backend
npm install
```

### Callout Max App
Navigate to the `callout-max-app` directory and install the dependencies:
```
cd callout-max-app
npm install
```

## API Documentation
- Endpoint to create a project: `POST /api/projects`
- Endpoint to fetch projects: `GET /api/projects`
